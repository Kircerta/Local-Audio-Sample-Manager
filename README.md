# LocalAudioSampleManager 说明文档

- 在Enter Keywords处输入搜索关键词，点击search搜索采样。
  - 点击下方出现的采样进行音频预览，再次点击则可中断音频播放
  - 右键采样出现选项 “Add to collection”
    - 选中后，列表中此采样前的名字会出现saved。这里并不会改变原始文件名里采样本身名字。
    - 上方可以选择搜索loop或者one shot
    - 可以搜索指定范围BPM的采样
      - 如果前后输入为同一个数字，则转而搜索精准BPM
- 采样的分类与索引基于文件本身的命名，本管理器并无分析音频的能力
- 选中采样后即可拖拽采样至宿主软件/其它位置，拖拽后的采样会被复制到新的位置。
